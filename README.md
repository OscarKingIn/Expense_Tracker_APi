## Daily Expense Tracker API

A simple REST API built with Python and Flask for adding and viewing daily expenses.  
This project demonstrates backend development, database handling, and API design.

## Features
- Add new expenses  
- List all expenses  
- SQLite database storage  
- Clean API routes  
- Lightweight and fast  

##  Tech Stack
- Python  
- Flask  
- SQLite  

##  How to Run
```bash
pip install -r requirements.txt
python app.py
